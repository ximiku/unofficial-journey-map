# Roadmap

## 已实现

- Vite + React + TypeScript 静态应用。
- 26 个路线节点、批注系统、审稿系统、数值系统、结局系统、档案馆。
- 10 个代表性互动模块。
- 克制古籍 / 档案视觉系统、响应式布局、presentation mode。
- 单元与组件测试。

## 近期改进

- 为更多节点补充专属互动模块。
- 增加可导出的最终路线报告图片。
- 将地图图层的差异做得更细：历史幽灵、地下妖洞和梦境层可拥有独立路径。

## 内容扩展

- 浪浪山支线。
- 白龙马梦境线。
- 法宝流通线。
- 妖怪后台线。

## 视觉打磨 backlog

- 更细的纸张折痕和印章盖落动画。
- 结局页面的仪式化地图转化。
- 更完整的打印样式和展板模式。

## 技术债

- 首版没有 e2e 测试，只做浏览器手动验收。
- 点击式批注交互可在后续升级为拖拽。
