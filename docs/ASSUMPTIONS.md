# 实现假设

- 本项目首版以课程展示为目标，不实现后端、多用户同步、真实 GIS 或远程图片资源。
- 批注放置使用点击式交互；拖拽可作为后续增强，不改变数据合同。
- 资料引用以章节范围和研究线索为主，不复制长篇原文。
- 小游戏采用 React 组件和少量 SVG/CSS 表达，不引入 Phaser。
- 视觉方向以文书档案和古籍卷轴为底，但允许适度放开：更明显的朱印、线绳、地图裂痕、纸片反馈和小妖档案员提示可以进入首屏，只要不牺牲阅读和操作清晰度。
- 地点推进采用“线性推进 + 可回看”：未来节点可以通过滑条远望但不可直接进入；默认起点仍为双叉岭，长安 / 法门寺作为序章在新游戏中视为已完成。
